### Finding a student account on the website {#finding-a-student-account-on-the-website}

1. This task is something you will actually spend a lot of time; there are often issues finding student accounts due to one of the following (1) Student has multiple accounts (2) Student forgot their username and provides only their name (3) Student registered under parent’s name (4) Student registered for the wrong type of account (ie. a parent account) (5) Student registered using different name or made a typo in their account name  
2. The steps to remedy these problems are fairly straight forward, but basically, you should:  
1) Start by searching the name provided  
2) If no name was provided, search the email address  
3) If the full name gives no results, try searching just their last name and scrolling through (do the same for the first name, too)  
4) Try searching their parent/guardian’s name  
5) Email the student/parent asking for a username  
6) One thing that happens sometimes is a school group will tell you a student is registered but you can’t find their account; one option for this is to use arbitrary user lists and filter by school (using only part of the school’s name to search)  
7) If you really can’t find the account, ask the student to make a new account

### Troubleshooting Student Errors {#troubleshooting-student-errors}

Very often, you will receive emails from parents/students complaining that some aspect of the website is not working. You should follow these steps or something similar to fix the issue:

1. Consider if the student should be able to do what they are trying to do (ie. if they are trying to pay for the program after the payment deadline, then they will obviously have trouble); this resolves a large chunk of the emails you will get  
2. Morph into the user and try and see if the error is reproducible  
3. Also test to see if the error is reproducible with the fake student account (there are a number of these, for example the account with username “fakestudent1” is a recent one that works)  
4. Try to pinpoint the cause of the error; for example, if the error is reproducible with the student account but not the fake student account, then there must be something weird in the student account that differs from the fake student account  
5. If it is reproducible and you don’t know how to fix it, contact others; if it isn’t reproducible then (a) give step by step instructions on how to perform the task and tell them that if they still are having difficulties we suggest the student/parent tries again after restarting their browser and clearing their cookies or using a different web browser (b) if needed, perform the task for the student (ie. if the student can’t order lunch, you can ask them what they want and do it for them; BUT, we try not to register students for classes on their behalf; this is because many classes fill quickly and it’s unfair to put kids into classes through our computers. The one exception to this is when we grant a student permission to take a higher level class (see Scenario 4 below))

### Adjusting account permissions {#adjusting-account-permissions}

Adjusting account permissions is useful in many scenarios, for example, if you want to test something with a fake student account, but registration isn’t open yet, you have to adjust the permissions of the fake student account to let them register early; **you should never** adjust the deadlines for Splash to achieve the same goal. Another example is when you want to let someone register late due to extenuating circumstances (ie. they tried to register at 4:59pm but the website crashed).

Find username of the account you’d like to give permissions

* Give this account permissions.  
  * Go [here](https://yale.learningu.org/admin/users/permission/)  
  * Add a new permission, giving the user the needed permissions. **Do NOT fill out the role field.** If you put student there, anyone with the role of student will have those permissions. That is NOT what you want (this would be equivalent to changing something on the deadlines page).  
* If you added a fake student to the program, make sure to go and remove this fake student from your program. To do this, go [here](https://yale.learningu.org/admin/program/studentregistration/) and search for the user. Select all the checkboxes relevant to your program, and press "expire student registrations". Also go [here](https://yale.learningu.org/admin/accounting/transfer/), search for the user, select the checkboxes relevant to your program, and press "delete selected transfers."

### The “Transfers” section of the website {#the-“transfers”-section-of-the-website}

The [transfers section](https://yale.learningu.org/admin/accounting/transfer/) is one place to go if you want to quickly find out the following:

1. Has a student paid  
2. Is a student on financial aid  
3. Did a student order lunch  
4. Did a student order a shirt

Some notes about Transfers: 

1. It has information for **all** programs; be sure that you’re only looking at the info for your program\!  
2. If the “Source” column says “Grants” it means the student is on Financial aid

### Grade change requests {#grade-change-requests}

You will sometimes receive emails that look like this:

| server@yale.learningu.org |  |  |  |
| :---- | :---- | :---- | :---- |

A grade change request was received from: Joshua Tochimani (ID: 22032, e-mail: joshuatoch95@gmail.com).

Current Grade: 8

Claimed Grade: 9

Reason: Miss clicked

Please review the request here: [http://yale.learningu.org/admin/users/gradechangerequest/243/](http://yale.learningu.org/admin/users/gradechangerequest/243/)

If this occurs, you should look at the reason, and if it seems legitimate, then approve the request. Occasionally, students will attempt to change their grade to try and get into a higher level class. While it’s impossible to tell for sure when this has happened, there are sometimes clues. For example, in Spring 19’, 3 kids from the same school all tried to change their grade from 8th to 9th because they had all “misclicked” within the span of 5 minutes. In those cases, you should send an email like the one below:

Hello,

We noticed that you submitted a grade change request from 8th to 9th grade. Please respond with evidence of your enrollment in 9th grade so that we can approve your request (this could be a class schedule or any other school-issued document that has your grade on it).

Thank you,

\-Splash Team
